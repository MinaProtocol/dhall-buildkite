MinaProtocol Dhall Base Package Releases

This is a base package for MinaProtocol Dhall Buildkite bindings which helps further more specific development for CI of other projects.

Each link on the page, for example 1.0.0/package.dhall, is an import page for version 1.0.0 which one can use in their own Dhall project by creating a new file with the following content:

https://s3.us-west-2.amazonaws.com/dhall.packages.minaprotocol.com/buildkite/releases/{version}/package.dhall sha256:{sha256}
